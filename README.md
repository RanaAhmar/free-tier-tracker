## 🎯 The Mission

Cloud providers change their pricing models constantly. It's frustrating to start building on a "free" tier only to realize you hit an invisible limit on day two.

This repository tracks **true** free tiers.
- No "free trial for 14 days"
- No hidden credit card charges
- Explicit details on limits (daily, monthly, rate limits)

## 🗂️ Categories

### ⚡ Serverless & Edge Compute
- **Vercel**: Next.js native, 100GB bandwidth, serverless functions.
- **Cloudflare Workers**: 100,000 requests/day, incredible edge execution.
- **Netlify**: Great for Jamstack, 100GB bandwidth, easy CI/CD.

### 🗄️ Databases (SQL & NoSQL)
- **Supabase**: 500MB database, 1GB storage, built-in Auth.
- **Neon**: Serverless Postgres, 500MB storage, generous compute.
- **MongoDB Atlas**: 512MB M0 cluster, unlimited time.

### 🧠 AI & Machine Learning APIs
- **Hugging Face**: Free inference API for thousands of open models.
- **Groq**: Insanely fast LPU inference (generous free rate limits).
- **Gemini (Google)**: Free tier for developers with high daily requests.

### 📧 Email & Notifications
- **Resend**: 3,000 emails/month (100/day).
- **SendGrid**: 100 emails/day forever.
- **Novu**: Open-source notification infrastructure (generous cloud free tier).

### 🔒 Authentication
- **Clerk**: 10,000 Monthly Active Users (MAU).
- **Auth0**: 7,000 active users.

## 🤝 How to Contribute

Has a provider changed their pricing? Did you find a hidden gem?
1. Fork the repository
2. Edit the list in `README.md`
3. Clearly state the exact limits of the new free tier
4. Submit a Pull Request!
